React To-do app
A responsive task management application built with React and Vite, featuring add, delete, and filter functionality with persistent storage via localStorage. Designed as a portfolio project to demonstrate front‑end development skills, clean UI design, and practical problem‑solving.
- Tech stack: React, Vite, CSS
- Key features: Task creation, deletion, filtering, persistent state
- Purpose: Showcases ability to build modern, interactive web apps for interview preparation and portfolio presentation
